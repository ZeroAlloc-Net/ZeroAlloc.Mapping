# ZeroAlloc.Mapping v1.2 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add 3 backlog items (B9 `[MappingCulture]`, B5 update-in-place, B2 polymorphic dispatch) to a v1.2 release of `ZeroAlloc.Mapping`.

**Architecture:** Each item extends existing v1/v1.1 components — `MapperDiscovery`, `PropertyMatcher`, `ConversionResolver`, `MapEmitter`, `TryMapEmitter`, `Diagnostics`. Three new opt-in attributes (`MappingCulture`, `PolymorphicMap<,>`, `PolymorphicTryMap<,>`); update-in-place opts in via user partial method declaration (no new attribute). Four new diagnostics (ZAMP012/013/014/015).

**Tech Stack:** .NET 8/9/10 multi-target, Roslyn `Microsoft.CodeAnalysis.CSharp` 4.14, xUnit 2, Verify 28.

**Step 2.4: Detect void-overload partial in `MapperDiscovery`**

Existing `FindUserPartialMethod` looks up `static (TSrc) → TDst`. Add a sibling helper:

```csharp
private static IMethodSymbol? FindUpdateInPlacePartial(INamedTypeSymbol owner, ITypeSymbol src, ITypeSymbol dst)
{
    foreach (var m in owner.GetMembers("Map").OfType<IMethodSymbol>())
    {
        if (!m.IsStatic) continue;
        if (!m.IsPartialDefinition && !m.PartialDefinitionPart != null) continue;  // any partial declaration form
        if (m.ReturnsVoid != true) continue;
        if (m.Parameters.Length != 2) continue;
        if (!SymbolEqualityComparer.Default.Equals(m.Parameters[0].Type, src)) continue;
        if (!SymbolEqualityComparer.Default.Equals(m.Parameters[1].Type, dst)) continue;
        return m;
    }
    return null;
}
```

(Note: the precise Roslyn API for "is partial" varies — `IsPartialDefinition` works in 4.7+. If compile fails, fall back to checking `m.DeclaringSyntaxReferences.Any(r => r.GetSyntax() is MethodDeclarationSyntax mds && mds.Modifiers.Any(SyntaxKind.PartialKeyword))`.)

Emission shape:

```csharp
public static partial void Map(global::Src src, global::Dst existingDst)
{

    global::System.ArgumentNullException.ThrowIfNull(existingDst);
```

For each `match.Mappings`, emit `existingDst.<TargetParamName> = <expr>;`. For `match.Constants`, emit `existingDst.<TargetParamName> = <constant>;`.

**Settable-property filter:** when generating update-in-place, the matcher walks ctor params (existing logic), but the generator must verify each matched property has a public setter on the destination type. If ANY matched param lacks a settable property → emit ZAMP012 in `MappingGenerator.ReportPerClassDiagnostics` (Step 2.7) and skip emission of this overload (so the user's compile error is just the unimplemented partial).

Helper to add (to MapEmitter or shared):

```csharp
private static bool DestPropertyIsSettable(INamedTypeSymbol dst, string paramName)
{
    var prop = dst.GetMembers(paramName).OfType<IPropertySymbol>()
        .FirstOrDefault(p => p.DeclaredAccessibility == Accessibility.Public);
    return prop?.SetMethod is { DeclaredAccessibility: Accessibility.Public, IsInitOnly: false };
}
```

If `DestPropertyIsSettable` returns false for any matched param when emitting update-in-place, ABORT this overload's emission (don't emit the method body; the user's `partial void Map(...)` declaration becomes an "unimplemented partial" — but the C# compiler accepts that as a no-op partial declaration).

Actually, that's wrong — an UNIMPLEMENTED partial method is one without an implementation body, which is a no-op call. The C# compiler doesn't error. So: if ZAMP012 fires, just don't emit the body. The user's call site silently becomes a no-op. This is acceptable v1.2 behaviour because ZAMP012 is an Error diagnostic — the compile fails before the user can ship.

```csharp
        polyKind = MappingKind.Map;
    else if (polymorphicTryMapAttr is not null && SymbolEqualityComparer.Default.Equals(orig, polymorphicTryMapAttr))
        polyKind = MappingKind.TryMap;
    else
        continue;

    var typeArgs = attr.AttributeClass!.TypeArguments;
    if (typeArgs.Length != 2) continue;
    if (typeArgs[0] is not INamedTypeSymbol baseType) continue;
    if (typeArgs[1] is not INamedTypeSymbol baseDstType) continue;

    polymorphics.Add(new PolymorphicDecl(
        BaseTypeFqn: typeArgs[0].ToDisplayString(SymbolDisplayFormat.FullyQualifiedFormat),
        BaseDestinationTypeFqn: typeArgs[1].ToDisplayString(SymbolDisplayFormat.FullyQualifiedFormat),
        Kind: polyKind,
        Location: type.Locations.FirstOrDefault() ?? Location.None,
        BaseTypeSymbol: baseType,
        BaseDestinationTypeSymbol: baseDstType));
}
```

Pass `PolymorphicDecls: polymorphics.Count > 0 ? polymorphics : null` to `MapperClass`.

`EmitPolymorphicDispatcher`:

```csharp
private static void EmitPolymorphicDispatcher(StringBuilder sb, PolymorphicDecl poly, MapperClass cls)
{
    // Find sibling [Map<TDerived, TDerivedDto>] decls where TDerived : TBase and TDerivedDto : TBaseDto.
    var cases = cls.Mappings
        .Where(m => m.Kind == MappingKind.Map)
        .Where(m => /* TDerived assignable to TBase via SymbolEqualityComparer or InheritsFromOrEqualsCheck */)
        .ToList();

    sb.Append("    public static ").Append(poly.BaseDestinationTypeFqn)
      .Append(" Map(").Append(poly.BaseTypeFqn).Append(" src)\n    {\n");
    sb.Append("        global::System.ArgumentNullException.ThrowIfNull(src);\n");
    sb.Append("        return src switch\n        {\n");
    for (int i = 0; i < cases.Count; i++)
    {
        var c = cases[i];
        sb.Append("            ").Append(c.SourceTypeFqn).Append(" __").Append(i)
          .Append(" => Map(__").Append(i).Append("),\n");
    }
    sb.Append("            _ => throw new global::System.InvalidOperationException(")
      .Append("\"No polymorphic mapping for runtime type \" + src.GetType().FullName)\n");
    sb.Append("        };\n    }\n");
}
```

The "TDerived assignable to TBase" check needs a helper:

```csharp
private static bool IsAssignableTo(INamedTypeSymbol derived, INamedTypeSymbol baseType)
{
    if (SymbolEqualityComparer.Default.Equals(derived, baseType)) return true;
    var cursor = derived.BaseType;
    while (cursor is not null)
    {
        if (SymbolEqualityComparer.Default.Equals(cursor, baseType)) return true;
        cursor = cursor.BaseType;
    }
    return derived.AllInterfaces.Any(i => SymbolEqualityComparer.Default.Equals(i, baseType));
}
```

Filter the cases by:
- Source: assignable to `poly.BaseTypeSymbol`
- Destination: assignable to `poly.BaseDestinationTypeSymbol`

Both ends must be checked — derived source MUST produce derived destination assignable to the base destination.

```csharp
public static void EmitPolymorphicTryDispatcher(StringBuilder sb, PolymorphicDecl poly, MapperClass cls)
{
    var resultType = "global::ZeroAlloc.Results.Result<" + poly.BaseDestinationTypeFqn + ", global::ZeroAlloc.Mapping.MappingError>";
    var cases = cls.Mappings.Where(m => m.Kind == MappingKind.TryMap)
        .Where(m => /* assignable check */)
        .ToList();

    sb.Append("    public static ").Append(resultType).Append(" TryMap(").Append(poly.BaseTypeFqn).Append(" src)\n    {\n");
    sb.Append("        if (src is null) return ").Append(resultType)
      .Append(".Failure(new global::ZeroAlloc.Mapping.MappingError(\"mapping.source.null\", \"(root)\"));\n");
    sb.Append("        return src switch\n        {\n");
    for (int i = 0; i < cases.Count; i++)
    {
        var c = cases[i];
        sb.Append("            ").Append(c.SourceTypeFqn).Append(" __").Append(i)
          .Append(" => TryMap(__").Append(i).Append("),\n");
        // Note: the leaf TryMap returns Result<TDerivedDto, MappingError>. The C# compiler's switch needs
        // an implicit conversion to Result<TBaseDto, MappingError>. If covariance isn't available, may need
        // explicit Map(value).Match(s => Result.Success(s as TBaseDto), f => Result.Failure(f)) — but
        // TBaseDto is a base class so the success-side cast is implicit reference conversion. Verify.
    }
    sb.Append("            _ => ").Append(resultType)
      .Append(".Failure(new global::ZeroAlloc.Mapping.MappingError(\"mapping.polymorphic.unhandled_type\", \"(root)\", \"runtime type \" + src.GetType().FullName + \" has no declared [TryMap]\"))\n");
    sb.Append("        };\n    }\n");
}
```

(The `Result<TDerivedDto, …>` to `Result<TBaseDto, …>` conversion is the trickiest part. If `Result<,>` is invariant in `T`, the implementer needs a cast or an explicit map. Verify `ZeroAlloc.Results.Result<,>`'s declaration. If invariant, emit:
```csharp
Cat __0 => TryMap(__0).Map<CatDto, AnimalDto, MappingError>(c => c)
```
or use the structural form. Inspect `Result<,>` API; if it has a covariant adapter, use it.)
